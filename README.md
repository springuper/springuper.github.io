# Shang Chun Github Pages

个人博客网站，使用 Hexo 构建，通过 GitHub Pages 部署。

## 快速开始

- **博客地址**: [https://springuper.github.io](https://springuper.github.io)
- **写作指南**: 查看 [BLOG_GUIDE.md](./BLOG_GUIDE.md) 了解如何添加新文章和发布流程

## 技术栈

- [Hexo](https://hexo.io/) - 静态博客框架
- [Even Theme](https://github.com/ahonn/hexo-theme-even) - Hexo 主题
- GitHub Pages - 静态网站托管
- GitHub Actions - 自动构建和部署
